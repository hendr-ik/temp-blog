About the banner on this page

Offen only collects data after opt in. This is done by clicking Yes please on the bottom banner entiteld Continue with transparent analytics.

If you do opt in your real usage data is added to the generated set of demo data.

Once you are opted in, you can opt out again in the Auditorium for users.

When you are opted out, you can simply delete all cookies for this page to display the banner again.
